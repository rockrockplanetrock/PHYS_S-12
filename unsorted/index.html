First, I used the exercise here(https://nathanmelenbrink.github.io/intro-dig-fab/09_networking/huzzah1.html) from class 09 where we turned on an LED using the internet on the Huzzah board. Just to backtrack, I downloaded the CP2104 USB Driver () for the Huzzah board () and pasted this URL (ttps://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json) into the Additional Boards Manager field within the ARduino IDE Preferences Menu. Afterwards, I added the "esp32" boards package within the Boards Manager under the Tools Menu in order to subsequently select the Adafruit ESP32 Feather as the board in use. Next, I opened the native example through File > Examples > Examples for Adafruit ESP32 Feather > WiFi > SimpleWifiServer. In the appropriate field, I replaced the fields specificed with my WiFi SSD and password. I then connected an LED to pin 5 (SCK), uploaded the program, and checked the serial monitor after hitting reset a few times to retrieve my IP address. It is imperative that the WiFi connection is 2.4GHz and not 5G! Once you have the IP address, paste it in your browser and you should see the following page. It will be responsive! Then, I uploaded this code (https://nathanmelenbrink.github.io/intro-dig-fab/10_machines/serial_setpoint_1.txt) and hooked up a stepper motor () and H-bridge to pins 11, 10, 9, and 8, as well as 3V and ground
